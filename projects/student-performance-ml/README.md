# 🎓 Student Performance Prediction

## 📌 Overview
This project predicts student exam scores based on study hours and sleep patterns using Machine Learning.

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## 📊 Workflow
1. Data Collection
2. Data Cleaning
3. Feature Selection
4. Model Training (Linear Regression)
5. Evaluation

## 🤖 Model
Linear Regression was used to understand the relationship between study habits and performance.

## 📈 Key Insights
- Study hours strongly impact scores
- Sleep plays a supporting role
- Balanced habits give better results
